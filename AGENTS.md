# Agents

This file defines the agents available for the Mementoes project.

## Available Agents

- **General**: For general software engineering tasks, including code editing, debugging, and implementation.
- **Explore**: For exploring codebases, searching files, and answering questions about the project structure.
- **Code**: For writing and refactoring code.

## Project Standards

- Follow React 19 + TypeScript best practices.
- Use Tailwind CSS for styling.
- Ensure accessibility and performance standards are met.
